# WeatherCompose
Weather compose is a project where I can demonstrate some of my skills using Jetpack Compose, kotlin and others libraries.

## Technologies
* Kotlin
* Jetpack Compose
* MVVM Architecture
  * ViewModel
  * ComposeState
  * StateFlow
  * Application
* Retrofit
  * REST API
  * GSON
* Dependency Injection
  * Dagger-hilt
* Coroutines
* Splashscreen
* Shimmer loading
* Empty state for no internet connection
* UI Compose Components
    * Box
    * Column
    * Row
    * Text
    * Image
    * Button
    * SearchBar
